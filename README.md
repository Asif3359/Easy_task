## Easy Task

Minimal personal portfolio layout built with Next.js and Tailwind CSS.

### Live Site

- https://easy-task-omega.vercel.app/

### Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS

### Getting Started (Local)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

### Notes

- Designed as a simple, single-page portfolio with sections for hero, resume, skills, portfolio, blog, pricing, and contact.
- Layout is responsive (mobile-first) and uses minimal custom styling on top of Tailwind utilities.
